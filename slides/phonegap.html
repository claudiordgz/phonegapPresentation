PhoneGap
Cordova

  • PhoneGap is a mobile development framework.
  • It runs on Apache Cordova... Cordova on the other hand is a mobile development platform AND framework.
  • It works as a server for you Web Applications
  • PhoneGap provides features of its own, but if your Web Application does not run on Cordova, it won't run on PhoneGap.
  • It Provides and interface to your sensors in your Native device.
